# 智能曲线合并快速参考

## ? 功能

自动检测并合并首尾相连的曲线

---

## ?? 使用方法

```
1. ?? 导入 STEP 文件
2. ?? 点击"合并曲线"按钮
3. ?? 查看检测结果
4. ? 确认合并
```

---

## ?? 快捷键

| 操作 | 快捷键 |
|------|--------|
| 合并曲线 | Ctrl+M |

---

## ?? 合并容差

| 容差 | 适用场景 |
|------|---------|
| 0.001 | 精密模型 |
| 0.01 | 一般模型（默认） |
| 0.1 | 粗糙模型 |

容差位置：左侧面板"合并容差设置"

---

## ?? 检测原理

系统检查 4 种连接：

```
1. 末端→起点 ●─────●─────
2. 末端→末端 ●─────  ─────●
3. 起点→起点 ─────●  ●─────
4. 起点→末端 ─────●  ●─────
```

---

## ?? 注意

```
?? 合并不可逆！原曲线会被删除
?? 建议先保存或备份
?? 合并后需要重新命名
```

---

## ?? 使用技巧

```
? 先检查检测结果
? 根据需要调整容差
? 不确定时先不合并
? 合并后点击"自动命名"
```

**问题**: 导入 66 条小线段  
**操作**: 点击"合并曲线"  
**结果**: 自动合并为 8 条完整曲线
