## ?? 使用方法

### 查看坐标系

```
1. 启动应用
2. 观察3D视图中心
3. ? 看到红/绿/蓝坐标轴
```

### 调整大小

```csharp
// 放大坐标系
pathEditor3D.UpdateCoordinateSystemSize(2.0);

// 缩小坐标系
pathEditor3D.UpdateCoordinateSystemSize(0.5);
```

### 显示/隐藏

```csharp
// 隐藏坐标系
pathEditor3D.SetCoordinateSystemVisibility(false);

// 显示坐标系
pathEditor3D.SetCoordinateSystemVisibility(true);
```

## ?? 注意事项

```
1. 坐标系位于原点 (0,0,0)
2. 如果数据不在原点附近，可能看不到
3. 建议使用 FitCurvesToView() 调整视图
4. 大场景时可能需要放大坐标系
```
