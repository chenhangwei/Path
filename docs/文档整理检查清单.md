### 旧文件处理

**方案A：归档（推荐）**
```powershell
# 运行归档脚本
.\archive-old-docs.ps1
```
这将把旧文件移动到 `docs_archive/` 目录

**方案B：直接删除**
```powershell
# 确认新文档无误后删除
Remove-Item "3D_APP_MIGRATION_GUIDE.md"
Remove-Item "3D_SNAP_INTEGRATION_GUIDE.md"
# ... 其他文件
```

### .gitignore 更新

如果选择归档，建议在 `.gitignore` 添加：
```
# 归档文档（可选）
docs_archive/
```
