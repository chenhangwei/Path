## ?? 常见操作示例

### 查看曲线细节

```
1. 鼠标移到感兴趣的曲线上
2. 向前滚动滚轮 2-3 次 (放大)
3. Shift + 中键微调位置
4. 中键旋转找最佳角度
```

### 对比多条曲线

```
1. 按 F 键看全景
2. 中键旋转到合适角度
3. 观察所有曲线的相对位置
```

### 检查放样结果

```
1. 执行放样 (Ctrl+L)
2. 按 F 键自适应新的点
3. 滚轮放大检查点分布
4. 中键旋转检查各个方向
```

## ? 专业提示

### ?? 精确控制
- **小幅度移动鼠标** = 精细控制
- **大幅度移动鼠标** = 快速导航

### ?? 流畅操作
- 不要快速抖动鼠标
- 平滑拖动体验更好
- 配合缩放使用效果最佳

### ?? 最佳实践
1. 先用 **F 键**看全局
2. 再用**中键**找角度
3. 最后用**滚轮+平移**看细节

## ?? 故障排除

### 问题: 按 F 键无反应
**解决**:
1. 点击 3D 视图区域获取焦点
2. 确认已导入曲线或有数据

### 问题: 中键旋转太快/太慢
**原因**: 这是正常的，灵敏度已优化

**建议**:
- 放慢鼠标移动速度
- 分多次小幅度调整

### 问题: Shift+中键平移方向反了
**这不是问题**: 
- 鼠标向右 = 场景向左（正常的 CAD 行为）
- 想象你在"推动"场景
