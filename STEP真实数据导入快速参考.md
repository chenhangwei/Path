```

   ↓
成功 → 曲线显示
```

## ?? 导入失败时

### 1. 查看错误信息

系统会显示详细原因和建议

### 2. 生成诊断报告

```
点击"是" → 自动生成报告 → 打开查看
```

### 3. 检查 CAD 导出

```
? 格式: STEP AP214
? 几何: 线框/曲线
? 确保包含曲线数据
```

## ?? CAD 导出关键设置

| CAD 软件 | 关键设置 |
|---------|---------|
| **CATIA** | 几何: Wireframe |
| **SolidWorks** | 输出为: Wireframe |
| **NX** | 几何选择: Curves |
| **Rhino** | Export Curves: Yes |

---

## ?? 常见问题

### "未找到曲线"

```
原因: 文件只有曲面/实体
解决: 导出时选择"线框"
```

### "解析失败"

```
原因: 文件格式不正确
解决: 使用 STEP AP214 格式
```
