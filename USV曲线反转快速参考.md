# USV 曲线反转快速参考

## ?? 功能简介

反转选中的 USV 曲线放样点的顺序，改变曲线方向。

---

## ?? 使用方法

### 方法 1: 快捷键
```
1. 选择一条已放样的曲线
2. 按 Ctrl+R
3. 确认提示对话框
```

### 方法 2: 菜单
```
1. 选择一条已放样的曲线
2. 菜单栏 → 放样(_L) → 反转曲线点顺序(_R)
3. 确认提示对话框
```

### 方法 3: 工具栏
```
1. 选择一条已放样的曲线
2. 点击工具栏上的 ?? 反转 按钮
3. 确认提示对话框
```

---

## ?? 快速示例

### 示例 1: 简单路径

**反转前**:
```
起点 A ──→ B ──→ C ──→ D 终点
     1     2     3     4
```

**反转后**:
```
终点 D ──→ C ──→ B ──→ A 起点
     12     3     4
```

### 示例 2: 曲线路径

**反转前**: 顺时针  
**反转后**: 逆时针

## ?? 完整工作流

```
1. 导入 STEP 文件 (Ctrl+I)
   ↓
2. 选择曲线
   ↓
3. 放样曲线 (Ctrl+L)
   ↓
4. 反转曲线 (Ctrl+R) ← 可选
   ↓
5. 生成步骤 (Ctrl+G)
```

---

## ?? 常见用途

### 1. 路径方向错误
**问题**: 曲线方向与预期相反  
**解决**: 使用反转功能

### 2. 创建对称路径
**问题**: 需要镜像路径  
**解决**: 放样 + 反转

### 3. 往复路径
**问题**: 需要来回路径  
**解决**: 正向 + 反向组合

```
放样(_L)
  ├─ 放样选中曲线(_S)   Ctrl+L
  ├─ 放样所有曲线(_A)   Ctrl+Shift+L
  ├─ ───────────────
  └─ 反转曲线点顺序(_R) Ctrl+R ← 新增
```

### 工具栏按钮
```
[?? 放样] [?? 放样全部] [?? 反转] ← 新增
```

---

## ?? 注意事项

1. **必须先放样**: 只有已放样的曲线才能反转
2. **自动更新**: 反转后 3D 视图自动刷新
3. **可重复**: 可以多次反转曲线
4. **影响生成**: 生成步骤时使用当前曲线方向
