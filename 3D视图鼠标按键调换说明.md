## ?? 使用场景

### 场景 1: 查看不同角度

```
1. 按住中键拖动 → 旋转模型
2. 从各个角度查看曲线和步骤
3. 松开中键停止旋转
```

### 场景 2: 移动视图位置

```
1. 按住右键拖动 → 平移视图
2. 将感兴趣的区域移到屏幕中心
3. 松开右键停止平移
```

### 场景 3: 放大查看细节

```
1. 滚轮向前滚动 → 放大
2. 查看曲线细节或 USV 标签
3. 滚轮向后滚动 → 缩小
```

## ?? 配置说明

### HelixToolkit 默认手势

HelixToolkit.Wpf 的默认鼠标手势配置：
```csharp
// 默认配置（内置）
RotateGesture = new MouseGesture(MouseAction.MiddleClick);  // 中键旋转
PanGesture = new MouseGesture(MouseAction.RightClick);   // 右键平移
ZoomGesture = new MouseGesture(MouseAction.WheelClick);     // 滚轮缩放
```

这是最常用的配置，符合大多数工程软件的习惯。
